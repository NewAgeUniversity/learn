Topic 1.2: Getting Started

Before diving into coding, it’s important to ensure that you have the right tools set up on your machine. Here are some prerequisites and tools you’ll need to get started:

1. Development Environment

  • Text Editor/IDE: A code editor like Visual Studio Code is essential for writing and editing code. It's lightweight but powerful, with support for extensions that make development easier. Other good alternatives include Sublime Text or Atom.
  • Web Browser: A modern browser such as Google Chrome or Mozilla Firefox will allow you to test and debug your web pages. Both come with developer tools that are critical for web development.

2. Version Control with Git

  • Git: Version control is a key aspect of web development. It allows you to track changes in your code, collaborate with others, and roll back mistakes. Git is the most popular version control tool, and throughout this course, we’ll use it to manage our projects.
  • GitHub: A platform to store your Git repositories. GitHub enables you to collaborate with others, showcase your work, and even deploy applications.

3. Command Line Basics

Learning how to navigate and execute commands in the command line (terminal) is essential for setting up development environments and managing projects. Throughout the course, we’ll be using the terminal to execute Git commands, install dependencies, and run servers.

4. Browser Developer Tools

Both Chrome and Firefox offer powerful developer tools. You'll use these to inspect elements, debug code, and test mobile responsiveness. We'll explore how to use these tools effectively as we progress through the course.

5. Basic HTML/CSS/JS Knowledge

While this course will start from the basics, having a fundamental understanding of how websites are structured and styled will be helpful. If you have no prior experience, don't worry— we'll cover everything from the ground up.

Now that you have everything ready, we’ll start by building a solid foundation in HTML and CSS, which will set the stage for more advanced topics. Let’s begin your journey into web development!
